里面都是静态方法

```java 
//判断给定的字符是否是大写
public static boolean isUpperCase(char ch)
//判断给定的字符是否是小写
public static boolean isLowerCase(char ch)
//判断给定的字符是否是数字字符
public static boolean isDigit(char ch)
//把给定的字符转成大写
public static char toUpperCase(char ch)
//把给定的字符转成小写
public static char toLowerCase(char ch)
```
